## Bagger 进度礼包插件
- 作者：[Soof4](https://github.com/Soof4)
- 修改：羽学
- 羽学给它做汉化并，加了个命令【/礼包 重置】  
- 方便重置服务器时清理它自带的数据库表
- 并支持配置更多可领取的物品

## 权限与命令
|权限  | 命令   | 说明 |
|-------------|-------------|-------------|
|bagger.getbags |/礼包 或 /pk|  查看或领取礼包指令菜单 |
|bagger.admin |/礼包 重置 | 重置礼包数据库 |
